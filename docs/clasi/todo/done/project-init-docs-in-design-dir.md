# Ensure project initiation puts documents into docs/clasi/design

The project initiation phase should output its documents (overview, specification,
use cases, etc.) into `docs/clasi/design/` rather than any other location. Verify
the `project-initiation` skill and any related tooling writes to the correct
directory, and fix if needed.

Sometimes the overview.md file is generated in the wrong location.
